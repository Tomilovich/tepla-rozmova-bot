# ☕ Чат-бот кав’ярні "Тепла Розмова"

## 📌 Опис
Телеграм-бот для замовлення кави з системою бариста.

## ⚙️ Функціонал
- замовлення кави
- кошик
- бонуси
- панель бариста
- статуси замовлення

## 🚀 Запуск

1. Встановити залежності:
pip install -r requirements.txt

2. Створити .env файл:
BOT_TOKEN=your_token
BARISTA_BOT_TOKEN=your_token
BARISTA_CHAT_ID=your_id

3. Запустити:
python main.py
python barista_bot/main.py
## 🚀 Як запустити проєкт

1. Клонувати репозиторій:
   git clone https://github.com/Tomilovich/tepla-rozmova-bot.git

2. Перейти в папку:
   cd tepla-rozmova-bot

3. Встановити залежності:
   pip install -r requirements.txt

4. Створити файл `.env`:
   BOT_TOKEN=your_token
   BARISTA_BOT_TOKEN=your_token
   BARISTA_CHAT_ID=your_chat_id

5. Запустити боти:

Клієнтський:
python main.py

Бариста:
python barista_bot/main.py
